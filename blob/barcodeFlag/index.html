European Barcode flag

  • Flag
    • Flag size follows a 2:3 proportion.
    • Inner space is divided in equal spaces, one for each country. There are as many spaces as are countries.
  • Size & Color
    • Everything is discarded but colors.
    • Each color bar is represented as a % of the total % presented in the official flag, with a minimum 5% width for any given color.
    • There's no need to place a color as a single bar, it can be splitted.
    • In case of not a plain color, the conflicting image is averaged to a plain color and then added.
    • It is desirable that smaller colors are placed inside, and big color areas are splitted to surround them.
  • Rotation
    • If the original flag is divided in vertical fields, there's no rotation.
    • If a flag needs to be rotated, the honor point (top left) is rotated vertically following the official rule of each country.
    • If there's no official rotation rule, the honor point (top left) still would be at the top left after rotation.
  • Color Collision
    • It is desirable to reorder flags where similar colors of different flags collide to improve separation.
  • Ordering
    • Flag is distinctive enough that any color ordenation will work.
    • If some country Brexites it's just as easily removed as added.
